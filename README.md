An enterprise-grade observability framework for Salesforce Platform Events.

## What This Solves
- Dead Letter Queue pattern for failed event processing
- High Volume Platform Event (HVPE) monitoring
- Multi-org event health dashboard (LWC)
- SIEM integration for enterprise security teams
